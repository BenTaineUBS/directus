# Notice

## Options

| Prop    | Description                   | Default  |
| ------- | ----------------------------- | -------- |
| `color` | Notice color                  | `normal` |
| `icon`  | Icon rendered before the text | `null`   |
| `text`  | Text for notice               | `null`   |

## Props

| Prop    | Description | Default                   | Type     |
| ------- | ----------- | ------------------------- | -------- |
| `color` |             | `'normal'`                | `String` |
| `icon`  |             | `'info'`                  | `String` |
| `text`  |             | `'No text configured...'` | `String` |
