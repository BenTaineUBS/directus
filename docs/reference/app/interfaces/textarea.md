## Props

| Prop          | Description | Default        | Type                                     |
| ------------- | ----------- | -------------- | ---------------------------------------- |
| `value`       |             | `null`         | `String`                                 |
| `disabled`    |             | `false`        | `Boolean`                                |
| `placeholder` |             | `null`         | `String`                                 |
| `trim`        |             | `true`         | `Boolean`                                |
| `font`        |             | `'sans-serif'` | `'sans-serif' or 'serif' or 'monospace'` |
